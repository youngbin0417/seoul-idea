알겠습니다. Figma로 구성도가 잘 보이는 목업(Mid-fidelity Wireframe)을 만들어드릴게요.

**핵심 화면 흐름**을 먼저 설계하고, 각 화면의 **구성 요소와 레이아웃**을 상세히 정리해 드리겠습니다.

## 전체 화면 흐름 (User Flow)

```
[온보딩/프로필 설정]
    ↓
[홈 (메인 대시보드)] ←→ [알림 센터]
    ↓
[활동 선택] → [추천 결과 (OAI 점수)] → [장소 상세] → [길찾기/예약]
    ↓                    ↓
[시간대별 예측 그래프]  [커뮤니티 피드/제보]
    ↓                    ↓
[체크리스트/가이드]    [행사 정보/도로상황 오버레이]
```

## 화면 1: 온보딩 / 프로필 설정 (최초 1회)

**목적**: 개인 맞춤 가중치 수집

| 영역 | 구성 요소 |
|------|----------|
| 상단 | 로고 + "맞춤 야외활동을 위해 알려주세요" |
| 섹션 1 | **선호 활동** (최대 5개): 그리드 아이콘 12개 (러닝, 자전거, 등산, 골프, 배드민턴, 수상스포츠, 캠핑, 피크닉, 아이동반, 반려견산책, 축구/야구, 야외촬영) |
| 섹션 2 | **건강/특성 체크**: 알레르기 보유, 영유아 동반, 고령자, 반려동물 동반 (토글) |
| 섹션 3 | **지역 설정**: "서울시 ▽" + 자치구 선택 (드롭다운) |
| 하단 | "시작하기" CTA 버튼 (Primary) |

**Figma 팁**: 아이콘은 48x48, 그리드 4열, 선택 시 Primary 컬러 fill + 체크 아이콘 오버레이

## 화면 2: 홈 (메인 대시보드)

**핵심**: 현재 위치 기반, 선택한 활동들의 실시간 OAI 한눈에 보기

```
┌─────────────────────────────────────┐
│  [햄버거메뉴]  서울시 강남구 ▼   [🔔] │
├─────────────────────────────────────┤
│  "오늘의 야외활동"                  │
│  2026.05.11 (월) 18:42             │
├─────────────────────────────────────┤
│  [활동 카드 수평 스크롤]            │
│  ┌─────────┐ ┌─────────┐ ┌────────┐│
│  │ 🏃 러닝  │ │ ⛳ 골프  │ │ 🐕 산책 ││
│  │   85점  │ │   62점  │ │  91점   ││
│  │  "최적"  │ │ "보통"  │ │ "최적"  ││
│  │ [상세보기]│ │ [상세보기]│ │[상세보기]│
│  └─────────┘ └─────────┘ └────────┘│
├─────────────────────────────────────┤
│  [지도 영역 - 60% 높이]              │
│  • 현재 위치 핀 (파랑)              │
│  • 추천 장소 핀들 (점수별 색상)      │
│    - 80~100: 초록                   │
│    - 60~79: 노랑                    │
│    - 40~59: 주황                    │
│    - 0~39: 빨강                     │
│  • 커뮤니티 핀 (작은 말풍선 아이콘)   │
│  • 행사 마커 (별 아이콘)             │
│  • 돌발 오버레이 (주황 점선 원)       │
├─────────────────────────────────────┤
│  [하단 탭바]                         │
│  [🏠홈] [🔍탐색] [📍커뮤니티] [👤마이]│
└─────────────────────────────────────┘
```

**Figma 구성**:
- **활동 카드**: 120x160, border-radius 16, 그림자 shadow-sm
- **점수 배지**: 원형 40x40, 점수 색상 fill
- **지도**: 카카오맵 API 임베드 영역 (Rectangle placeholder 375x500)
- **핀 컴포넌트**: 24x24 아이콘 + 8x8 점수 도트 (우측상단)
- **레이어 토글 FAB**: 우측 하단 둥근 버튼 3개 (커뮤니티/행사/돌발 ON/OFF)

- **아코디언**: 44px 헤더 + 펼침 시 콘텐츠 영역
